# 🐾 Tinder Pet (Frontend)

Приложение в стиле Tinder для животных: свайпы, лайки, мероприятия и чаты.

---

## 🚀 Возможности

- 🐶 Лента животных (свайпы)
- ❤️ Лайки
- 👤 Профиль пользователя
- 🐾 Управление своими питомцами
- 📅 Мероприятия
- 💬 Чаты

---

## 🛠 Технологии

- React
- TypeScript
- Zustand (state management)
- TailwindCSS
- Vite

---

## ⚙️ Запуск проекта

### 1. Установить зависимости

```bash
npm install
```
### 2 Запуск, но только на локальной версии с локальным бэком:
```bash
npm run dev 
```
